Below is a gif of the remote update feature:

This feature is similar to the Touchpad for iPhones. This app follows this user flow:

User opens app.

User logs into app via fingerprint.

App loads user profile settings.

If update is necessary, then app prompts user for update.

User starts update from any location.

This feature allows the user to remote update their PS4 from anywhere so that they can minimize wait time for game play.

Reflection

I enjoyed thinking outside of the boundaries to create something new. My benchmark for this app included iPhones and remote car starters. For Sony to implement this feature, they would need to work with electrical engineering teams. Ultimately, this design challenged improved my ability to benchmark features from other products and recreate them into existing designs.
