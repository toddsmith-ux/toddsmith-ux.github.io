Phase Five: Final Design

Redesign

My redesign utilizes the white space as a border around the grid. The colour palette uses shades of blue to convey a tranquil, relaxing atmosphere on the site. The three colours in the banner represent the three components of the triangle. The font style is primarily Century; this font reminded me of a classic early 1900s vintage newspaper style.

Figure 6: Final Home Page Panel

For the navigation, this redesign uses six main options that users can click to find their desired information. These categories aspire to differentiate between each other so that users can know which path to start.

My primary design inspiration was the New York Times. The New York Times maintains a grid format across its homepage. Google’s centered search bar inspired this design. Also, Chanel’s minimalist design also inspired this design. Furthermore, the minimalist focus derives from these websites.

Conclusion

Overall, I learned from this project that audience analysis is one of the most important factors in user interface design. Sites, such as WRAL, must appeal to a wide array of users. Thus, they must constantly conduct user testing to determine if there are new audience demographics or if their audience’s needs changed. Furthermore, this project further solidified that testing is an iterative process.
