PS4 App Redesign

This redesign was a response to a design thinking challenge for class. I focused on the primary question: “how might we reduce the amount of time that PS4 users spend waiting for games to update?” To answer this question, I proposed a PS4 app redesign that enabled users to remote start game updates. This project occurred in Fall 2019. To learn more about this project, visit this page.
